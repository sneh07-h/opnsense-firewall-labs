# 🔥 OPNsense Firewall Labs

## Overview

This repository contains hands-on OPNsense Firewall lab exercises completed using VirtualBox, Kali Linux, and Ubuntu Server.

## Labs Included

### 📘 Lab 01 – OPNsense Installation & Initial Configuration
- OPNsense Installation
- WAN & LAN Configuration
- DHCP Setup
- Internet Connectivity
- Web GUI Configuration

### 📘 Lab 02 – Firewall Rules & ICMP Filtering
- Firewall Aliases
- ICMP Filtering
- Rule Order Evaluation
- Google DNS Blocking

### 📘 Lab 03 – Network Segmentation & Firewall Policy Validation
- Network Segmentation
- OPT1 Configuration
- SSH Rules
- SMB/Telnet Blocking
- Firewall Logging
- Troubleshooting

## Technologies Used

- OPNsense Firewall
- Kali Linux
- Ubuntu Server
- Oracle VirtualBox
- TCP/IP Networking
- Firewall Rules
- Network Security
